Compass

Your developer
workflow, in one place

Compass is a macOS app that brings together GitHub pull requests, Cursor workspaces, and Claude agent sessions — so you spend less time context-switching and more time shipping.

Download for Mac

Version 1.0.3 · macOS 15 Sequoia or later

Pull Requests

Track open PRs, view CI status, read comments, and get AI-powered code reviews — all without leaving the app.

Cursor Workspaces

Browse and open your Cursor workspaces tied to local repos with a single click.

Claude Agents

Explore your Cursor and Claude Code agent session history and transcripts in a searchable timeline.

Home Map

A canvas-style home that clusters your PRs, bookmarks, and agent sessions into one spatial overview.

Requirements

  • macOS 15 Sequoia+
  • gh CLI (authenticated)
  • Claude CLI
  • Cursor
  • iTerm2
